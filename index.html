About

Word Hunter is a simple "find-the-words" game that was originally forked from word-search-game by lizhineng.

The current game is still under development and we would love to see people help contribute to making this otherwise boring game into an enjoyable and fun game for the whole family.


Forked by rgloginja1 from RMG Software Pty Ltd on Saturday 17th January 2015.

For more information, please visit our GitHub repo.

New Game

Are you ready to see if you will make it to the Word Hunter leaderboard?

Hold your mouse over each button for a brief description of your goal for each game and click that button to start your game.

NB. Word Hunter not working on your mobile? This game was originally designed for use with browsers on PC's and laptops. We will be making an effort to make this work on all devices, including mobile devices, in the near future.
